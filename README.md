# Excel Worksheet Password Remover (VBA Version)

ابزار حرفه‌ای برای حذف رمز محافظت شیت‌های فایل‌های اکسل (پشتیبانی از فرمت‌های XLS و XLSX)

## ⚠️ سلب مسئولیت (Disclaimer)

**این ابزار فقط برای مصارف آموزشی و تست امنیتی روی فایل‌هایی که مالکیت آنها را دارید یا اجازه کتبی دارید، طراحی شده است.**

استفاده از این ابزار برای دسترسی غیرمجاز به فایل‌های دیگران غیرقانونی است. نویسنده هیچ مسئولیتی در قبال استفاده نادرست از این کد ندارد.

## ✨ ویژگی‌ها

- 🔓 **شکستن رمز همه شیت‌ها** به صورت خودکار
- ⚡ **سرعت بالا** با الگوریتم بهینه‌شده (حدود 10 برابر سریعتر از روش‌های معمول)
- 📊 **گزارش کامل** با ذخیره رمزهای پیدا شده در شیت جداگانه
- ⏱️ **نمایش زمان** اجرا برای هر شیت و کل عملیات
- 🎨 **رنگ‌بندی نتایج** (سبز: موفق، قرمز: ناموفق)
- 📋 **پشتیبانی از هر دو فرمت** XLS و XLSX
- 🔄 **بررسی خودکار همه شیت‌ها** (حتی شیت‌های مخفی)

## 📋 پیش‌نیازها

- Microsoft Excel 2007 یا بالاتر (نسخه‌های 32-bit و 64-bit)
- فعال بودن قابلیت ماکرو (Macro)
- ویندوز (پشتیبانی محدود از Mac)

## 🚀 نحوه استفاده

### روش سریع (کپی و اجرا)
- در ابتدا کد را حتما کپی کنید
1. فایل اکسل قفل شده را باز کنید
2. کلیدهای `Alt + F11` را فشار دهید (باز شدن ویرایشگر VBA)
3. از منوی `Insert` گزینه `Module` را انتخاب کنید
4. کد را در این بخش Past کنید و اجرا را بزنید
5. اجرا را بزنید و منتظر بمانید تا انجام شود
